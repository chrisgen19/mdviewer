# Writing Markdown

Learn how to write beautiful markdown documents for MDView.

## Headers

Use `#` symbols to create headers:

```markdown
# H1 Header
## H2 Header
### H3 Header
```

## Lists

Create bullet lists with `-` or numbered lists with `1.`:

- First item
- Second item
- Third item

1. Numbered item one
2. Numbered item two
3. Numbered item three

## Blockquotes

## Code

Inline code uses backticks: `const x = 10`

Code blocks use triple backticks:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Emphasis

- **Bold text** with `**text**`
- *Italic text* with `*text*`

Happy writing!
