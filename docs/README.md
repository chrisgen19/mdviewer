# Welcome to MDView

This is a modern, fast, and intuitive Markdown viewer built with Next.js.

## Features

- **Real File System Navigation**: Browse actual folders and .md files from your project
- **Dark Mode**: Toggle between light and dark themes for comfortable reading
- **Instant Rendering**: Fast markdown rendering with no lag
- **GitHub-Style Interface**: Familiar navigation patterns similar to GitHub

## Getting Started

1. Add your markdown files to the `docs` folder
2. Organize them in folders as you like
3. Navigate through them using the sidebar or file list

Enjoy exploring your documentation!
