# Installation Guide

Welcome! This guide will help you get started with MDView.

## Prerequisites

Before you begin, ensure you have the following installed:
- Node.js version 18 or higher
- npm or yarn package manager

## Installation Steps

1. Clone the repository
2. Install dependencies
3. Run the development server

```bash
npm install
npm run dev
```

## Configuration

The application reads markdown files from the `docs` folder by default. You can add your own markdown files and organize them in any folder structure you like.

## Next Steps

Once installed, you can:
- Add your markdown files to the docs folder
- Create subfolders to organize your content
- Start the dev server and browse your files
